# i.M.A.C iOS 團隊

致力於iOS應用程式開發，除了專注於專案的投入，也利用空餘時間製作iOS的元件、功能測試、SDK整合等，分享給同樣為iOS的開發者們。

# 參與貢獻

本團隊為台中科技大學-資訊工程系i.M.A.C社群，團隊致力於輔導學弟以及模組的製作，若有意願一起貢獻模組讓大家討論與參考，都可以聯絡本團隊。
